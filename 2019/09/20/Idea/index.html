The selected directory is not a valid home for JDK解决办法

module删除

右键->Open Module Settings->-号->OK->右键delete

模板

  • psvm/prsf/psf(i/s)
  • main方法/私有静态常量/public static final(int/string)
  • sout/soutp/soutm/soutv/xxx.sout
  • 输出/输出参数/输出方法/输出变量(就近)/输出xxx
  • fori/iter/itar
  • list.for/list.fori/list.forr(倒叙遍历)
  • ifn/inn/xxx.nn/xxx.null

    具体参考settings->Live Templates(可修改)
    或settings->General->Postfix Completion(不可修改)

  • Templates中psfi/psfs添加$VAR1$ = $VAR2$;
    $END$($VAR$光标位置)
